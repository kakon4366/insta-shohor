Question and Answer

1. How does Javascript works?

Ans: Javascript একটি Interpreted Programming Language। এটি লাইন বাই লাইন code read করে। Javascript Node.js বা ব্রাউজারের সাহায্যে run করে। Javascript ব্রাউজারের V8 নামক Engine এর সাহায্যে parser করে। তারপর সেই code কে convert করে মেশিন code এ রুপান্তর করে। তারপর ব্রাউজারের মাধ্যমে output দেখায়।

2. What does Javascript event loop?

Ans: Event loop Asynchronous পদ্ধতিতে চলে। Javascript এর execution stack এর সমস্ত code কে run করায়। সেখানে web APIs এর অংশ যেমন: setTimeout(), setInterval() ইত্যাদি। এগুলোর একটা Parameter হিসাবে Timer থাকে। সেই Timer background এ continue করে message queue নামক স্থানে থাকে। যখন Timer এর সময় শেষ হয় তখন সেটা execution stack এ গিয়ে run করে এবং সেটার output দেখায়। এরকম যতগুলো Asynchronous code পায় সবগুলোকে message queue এ সিরিয়ালি রাখে এবং Time count যার আগে শেষ হয় তাকে execute করে।

2. What is different localstorage vs sessionstorage?

Ans: localStorage: localstorage 10MB এর সাইজ হয়ে থাকে। localstorage এ কোন data রাখলে সেই data মুছে যায় না। অর্থাৎ browser close বা tab close করলেও সেই data save থাকে। এবং localstorage এর data অন্য tab এ use করা সম্ভব। sessionStorage: অন্যদিকে sessionstorage 5MB এর সাইজ হয়ে থাকে। sessionstorage এ কোন data রাখলে সেই data browser close বা tab close করলে আর ফিরে পাওয়া যায় না। localstorage এর মতো sessionstorage এর ডাটা অন্য কোন tab এর use করা যায় না।
